┌──────────────────────────────────────────────────────────────────────────┐
│ 🎯 QUÉ HACE ESTE COMPONENTE                                              │
└──────────────────────────────────────────────────────────────────────────┘

Un flujo interactivo de ONBOARDING en 5 pasos para recopilar información
sobre el negocio del usuario:

   📝 PREGUNTA 1: ¿Cómo se llama tu negocio y qué vende?
   🎭 PREGUNTA 2: ¿Cómo quieres que te perciban? (cercano/profesional/divertido)
   👥 PREGUNTA 3: ¿Quiénes son tus clientes principales?
   ⏰ PREGUNTA 4: ¿Horario de atención y cómo contactarte?
   🏆 PREGUNTA 5: ¿Cuáles son tus competidores principales?

   ⬇️

   💾 GUARDADO AUTOMÁTICO EN LOCALSTORAGE (JSON persistente)

   🔗 BOTÓN FINAL: "Conectar cuentas" (con callback)

const handleOnboardingComplete = async (data) => {
  try {
    // Guardar en backend
    await api.post('/businesses/setup', {
      nombre_negocio: data.nombre,
      tono_comunicacion: data.tono,
      publico_objetivo: data.publico_objetivo,
      horario_atencion: data.horario_contacto,
      competidores: data.competidores,
    });
    
    // Redirigir al dashboard
    navigate('/dashboard');
  } catch (error) {
    console.error('Error:', error);
    alert('Error al guardar datos');
  }
};
